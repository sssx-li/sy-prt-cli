## sy-prt-cli

集成项目开发模板 sy-prt-cli


### 所有命令
```
  帮助:
      sy --help

  查看版本:
      sy -V 或 sy --version

  查看模板列表:
      sy list 或 sy ls
  
  查看是否最新版本:
      sy upgrad 或 sy u

  创建项目:
      sy create <project>
  
  添加模板:
      sy add 或 sy a
  
  删除模板:
      sy delete 或 sy del

```


### 安装

```
  npm i sy-prt-cli -g
```

### 创建项目

```
  sy create <project>
```

### 依赖安装

```
  pnpm i
  or
  npm i
```

### 启动项目
```
  pnpm run dev
  or
  npm run dev
```
